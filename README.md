# 🌐 IP Subnetting & VLSM Tool

Una aplicación web ligera y eficiente creada con **Vanilla JavaScript**, diseñada para calcular subredes IPv4 de forma rápida y sencilla. Ideal para estudiantes de redes, ingenieros y administradores de sistemas que necesitan realizar cálculos de direccionamiento sin depender de calculadoras externas complejas.

## 🚀 Características

*   **Análisis de Red:** Obtén rápidamente el total de hosts utilizables y el tamaño del bloque a partir de una dirección IP y su máscara.
*   **Subnetting Fijo:** Calcula el número de bits necesarios para dividir una red en subredes de igual tamaño.
*   **VLSM (Variable Length Subnet Masking):** Optimiza tu espacio de direccionamiento ingresando la cantidad de hosts necesarios para cada segmento; la herramienta calculará automáticamente la máscara ideal para cada uno, evitando el desperdicio de direcciones.

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica.
*   **CSS3:** Diseño responsivo y moderno con enfoque en *Mobile-First*.
*   **Vanilla JavaScript:** Lógica de cálculo pura (sin frameworks ni dependencias externas).

## 📋 Cómo usarlo

1.  Clona este repositorio o descarga el archivo `index.html`.
2.  Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge, Safari).
3.  **Para Análisis:** Introduce tu IP y máscara base.
4.  **Para Subnetting Fijo:** Introduce el número de subredes que deseas crear.
5.  **Para VLSM:** Introduce la cantidad de hosts separados por comas (ejemplo: `50, 20, 10, 2`) y obtén el prefijo necesario para cada uno.
